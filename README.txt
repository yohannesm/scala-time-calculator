Assumptions: The hours passed will be from 0-23. 
Midnight or hour 24 will be counted a 0th hour of AM. 
Minutes being added in AddMinutes should always be positive. 
The minutes part of the clock in the string will need to be from 0-59

To run this program , run the scala compiler:
> scalac TimeCalc.scala

and then run the program with scala interpreter:
> scala TimeCalc

I put the test cases in main, since I didn't want to use a unit test framework but I could set those up too, it would just take more time.
